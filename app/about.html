PlexTogether is a tool to sync Plex content with friends and family anywhere in the world.

PT started off as Python script which we distributed amongst our friends. With a command line based UI we quickly realised this was too difficult for our friends to use reliably.

After playing around with the concept we decided to make a version that we could release that all Plex users could enjoy.
